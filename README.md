## Prerequisites

* Docker 19.03.x
* Docker Compose 1.25.x

## Installation

```sh
docker-compose up -d
```


## Used Technologies

* Spring Boot 2.3.5
* Spring Boot Web
* Spring Boot Actuator
* Lombok
* Dev Tools
